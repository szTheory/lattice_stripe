**Common pattern for ALL 6 files:**
- `use ExUnit.Case, async: false` (shared network resource)
- `import LatticeStripe.TestHelpers`
- `@moduletag :integration`
- `setup_all` block that checks stripe-mock connectivity via `:gen_tcp.connect(~c"localhost", 12111, [], 1000)`, starts `{Finch, name: LatticeStripe.IntegrationFinch}` via `start_supervised!`, or returns `{:skip, "stripe-mock not running on localhost:12111"}` if connection fails
- `setup` block that creates client via `test_integration_client()`
- Assert on struct types (`%Customer{}`, `%PaymentIntent{}` etc.) and that `id` is a non-nil binary (`assert is_binary(result.id)`)
- Do NOT assert on specific ID prefixes (stripe-mock IDs vary) -- assert `is_binary` and not nil
- Include one error case per file: retrieve with `"nonexistent_id_999"` expects `{:error, %Error{type: :invalid_request_error}}`
