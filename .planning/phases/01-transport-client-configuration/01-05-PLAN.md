4. **request/2** per D-04, D-16: Takes `(%Client{}, %Request{})`, returns `{:ok, decoded_body} | {:error, %Error{}}`.
   
   Steps inside request/2:
   a. Build full URL: `client.base_url <> request.path`
   b. Build headers list:
      - `{"authorization", "Bearer #{effective_api_key}"}` -- use per-request api_key from request.opts if present, otherwise client.api_key
      - `{"content-type", "application/x-www-form-urlencoded"}` for POST/PUT/PATCH methods
      - `{"stripe-version", effective_api_version}` -- use per-request stripe_version from request.opts if present, otherwise client.api_version
      - `{"user-agent", "LatticeStripe/#{Mix.Project.config()[:version]} elixir/#{System.version()}"}` -- store version as module attribute at compile time, NOT call Mix.Project at runtime
      - `{"stripe-account", stripe_account}` only if stripe_account is non-nil (from per-request opts or client default)
      - `{"idempotency-key", key}` only if provided in request.opts
      - `{"accept", "application/json"}`
   c. Build body:
      - For POST/PUT/PATCH: `FormEncoder.encode(request.params)`
      - For GET/DELETE: nil (params go as query string appended to URL)
   d. For GET/DELETE with params: append `?#{FormEncoder.encode(request.params)}` to URL
   e. Handle `expand` from request.opts: merge `expand` params into the request params as `expand[0]=..., expand[1]=...`
   f. Build transport opts: `[finch: client.finch, timeout: effective_timeout]` where effective_timeout comes from per-request opts or client default
   g. Wrap transport call in `:telemetry.span/3` (per D-16):
      - Event prefix: `[:lattice_stripe, :request]`
      - Start metadata: `%{method: request.method, path: request.path}`
      - On success: decode body via `client.json_codec.decode!/1`, check status:
        - 2xx: return `{:ok, decoded_body}` with stop metadata including status and request_id
        - 4xx/5xx: parse error via `Error.from_response/3`, return `{:error, error}` with stop metadata
      - On transport error: return `{:error, %Error{type: :connection_error, message: inspect(reason)}}` with stop metadata
      - If `telemetry_enabled` is false, skip the span and call transport directly
   h. Extract `request_id` from response headers: find header named `"request-id"` (case-insensitive)

   **Important User-Agent note:** Do NOT call `Mix.Project.config()` at runtime -- Mix is not available in production releases. Instead, store the version as a module attribute:
   ```elixir
   @version Mix.Project.config()[:version]
   ```
   And use `@version` in the header building function.

5. **@moduledoc** with clear examples:
   ```
   client = LatticeStripe.Client.new!(api_key: "sk_test_...", finch: MyApp.Finch)
   request = %LatticeStripe.Request{method: :get, path: "/v1/customers/cus_123"}
   ```
