6. Public API functions — each builds `%Request{}`, calls `Client.request/2`, unwraps via private helpers:
   - `create(client, params \\ %{}, opts \\ [])` — POST `/v1/customers`, unwrap_singular
   - `retrieve(client, id, opts \\ [])` — GET `/v1/customers/#{id}`, unwrap_singular, guard `when is_binary(id)`

   - `delete(client, id, opts \\ [])` — DELETE `/v1/customers/#{id}`, unwrap_singular, guard `when is_binary(id)`
   - `list(client, params \\ %{}, opts \\ [])` — GET `/v1/customers`, unwrap_list
   - `search(client, query, opts \\ [])` — GET `/v1/customers/search`, params `%{"query" => query}`, unwrap_list (typed items per Research Q1 recommendation), guard `when is_binary(query)`
   - `stream!(client, params \\ %{}, opts \\ [])` — builds Request, pipes `List.stream!(client, req) |> Stream.map(&from_map/1)`
   - `search_stream!(client, query, opts \\ [])` — builds search Request, pipes `List.stream!(client, req) |> Stream.map(&from_map/1)`, guard `when is_binary(query)`

7. Bang variants for each CRUD + search function: `create!/3`, `retrieve!/3`, `update!/4`, `delete!/3`, `list!/3`, `search!/3` — each calls non-bang and pipes through `unwrap_bang!/1`.

8. Private helpers:

   - `unwrap_list({:ok, %Response{data: %List{} = list} = resp})` -> `{:ok, %{resp | data: %{list | data: Enum.map(list.data, &from_map/1)}}}`
   - `unwrap_list({:error, %Error{}} = error)` -> `error`
   - `unwrap_bang!({:ok, result})` -> `result`
   - `unwrap_bang!({:error, %Error{} = error})` -> `raise(error)`
   - `from_map(map)` — maps all 26 known fields from string keys, `extra: Map.drop(map, @known_fields)`
